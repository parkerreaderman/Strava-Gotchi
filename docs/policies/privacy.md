# Sporty Gotchi Privacy Policy

_Last updated: {{DATE}}_ (update before launch)

1. **Data we collect**
   - Strava profile basics (name, profile photo) for personalization.
   - Activity summaries for the past 90 days (distance, duration, heart rate, power).
   - No location streams, private notes, or friend data.
2. **How we use it**
   - Calculate training metrics (CTL/ATL/TSB) to drive the avatar.
   - Cache recent activities locally so the dashboard loads quickly.
   - Never sell data or share with third parties.
3. **Storage & retention**
   - Tokens + activity caches live in our database (currently SQLite / Postgres).
   - Delete all user data within 30 days of disconnecting Strava (email support@sportygotchi.app).
4. **Security**
   - Secrets managed via Vercel env vars.
   - TLS enforced end-to-end (Vercel HTTPS).
5. **Your choices**
   - Disconnect via Settings → “Disconnect Strava”.
   - Request deletion via email; we confirm within 72 hours.
6. **Contact**
   - privacy@sportygotchi.app
